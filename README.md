# ☕ Cafe Menu — Web Page

> A clean, responsive cafe menu page built with pure HTML & CSS. No frameworks. No dependencies. Just good fundamentals.

🌐 **Live Demo:** [cafe.gulshankumar.com](https://cafe.gulshankumar.com)

## ✨ Features

- **Structured menu layout** — clear item and price alignment for easy reading
- **Responsive design** — adapts gracefully across screen sizes
- **Privacy-friendly fonts** — uses [Bunny Fonts](https://fonts.bunny.net/) instead of Google Fonts
- **Optimized images** — background uses WebP with JPG fallback
- **Semantic HTML** — improves accessibility and SEO
- **Hover effects** — subtle interactivity for a polished feel
- **Clean UI** — balanced spacing and purposeful design decisions

## 🎯 Purpose

This project was built to sharpen skills in:
- Layout precision with CSS (Flexbox / Grid)
- Core CSS fundamentals — specificity, box model, positioning
- Real-world UI thinking — readability, hierarchy, spacing
- Keeping code simple, clean, and maintainable
