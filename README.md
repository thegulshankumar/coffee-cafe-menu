## ☕ Cafe Menu Web Page

🌐 **Website URL:** https://cafe.gulshankumar.com

A clean and responsive Cafe Menu Web Page built using pure HTML and CSS. This project focuses on strong layout structure, readability, and practical UI design without using any frameworks.

---

### 🚀 Features

🔸 Structured menu layout with clear item and price alignment  
🔸 Responsive container for better viewing across devices  
🔸 Bunny Fonts for improved typography, consistent rendering, and better privacy
🔸 Background image fallback using WebP and JPG  
🔸 Semantic HTML for better accessibility  
🔸 Hover effects for interactive elements  
🔸 Clean spacing and balanced UI design  

---

### 🎯 Objective

To practice layout precision, core CSS fundamentals, and real-world UI thinking while keeping the code simple and maintainable.
